### 한길의 차별화 (Positioning)

PLUS에 **없는** 언어들 = 한길의 진입점:
- 신할라어 (스리랑카) ← 1차 출시
- 타갈로그어 (필리핀)
- 미얀마어, 캄보디아어, 몽골어, 우즈베크어, 네팔어, 인도네시아어, 태국어 등

이 사람들은 **현재 모국어로 공부할 앱이 전혀 없음.** → 무경쟁 시장.

### 사용자 흐름 (메인앱)
```

  ↓ 자동
[첫 실행만] 언어 선택 (신할라어 / 한국어)
  ↓
홈 (라이트 크림)
  ├─ 📖 순차 학습 (1번부터 997번)         ← MVP
  ├─ 🏆 명예의 전당 (다크+오로라)         ← MVP, 자동 마케팅

  ├─ 🔥 급할때 200문제                    ← 3차
```

## 8. 어제의 교훈 (작업 원칙)

> 미니앱 만들면서 배운 것 — 메인앱에서도 동일하게 적용

1. **단계별 체크포인트**: 한 번에 한 기능만, 작동 확인 후 다음
2. **작은 코드부터 → 작동 확인 → 다음 추가**
3. **옛 코드 위에 새 코드 얹지 말기** (덧붙이지 말고 교체)
4. **본인이 이해 못 하는 코드는 추가 안 함** (Claude가 설명 책임)
5. **iframe 이슈 주의**: Google Apps Script는 sandboxed iframe — `window.top.location.href` 필요
6. **권한은 점진적으로 추가**: getFolderById 권한 OK 받았어도 createFile은 또 따로 권한 필요
7. **테스트 시 시크릿 창**: 캐시 문제 회피
8. **새 코드 작성 = 새 배포**: Apps Script는 코드 수정해도 배포 안 하면 반영 안 됨
